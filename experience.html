Paytree

Mar 2022 – Oct 2024

Kotlin Flutter React.js

At Paytree, I developed a payment platform mostly used on events like festivals. This platform included:

  • A dashboard for product and cash register management, and statistics;
  • A mobile POS Android app used by the cashiers, where they can select products and pay using various payment methods, including NFC balance cards and contactless card payments;
  • A customer-facing balance card website, where users can see their balance and past orders, and top-up their balance cards using iDeal;
  • And an API that powers all of this.

The Paytree dashboard is built with Material UI for styling as well, but on top of plain React.js for now. I plan to move this to Next.js as well in the future.

The mobile POS app is built with Flutter.

The API that powers both of these is built using Ktor, Koin, Exposed, and backed by a PostgreSQL database.

All of this is deployed to and built using Google Cloud.
