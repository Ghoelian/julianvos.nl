Paytree

Mar 2022 – Oct 2024

Kotlin Flutter React.js

At Paytree, I built a payment platform designed for use at events such as festivals. The platform consisted of:

  • A web dashboard for product and register management, as well as sales insights and statistics;
  • A mobile POS Android app for cashiers, supporting product selection and multiple payment methods, including NFC balance cards and contactless payments;
  • A customer-facing website where users could view balances and order history, and top up cards via iDeal;
  • And a central API powering all components.

The dashboard is built with React.js and Material UI. The POS app is developed in Flutter, while the backend API is implemented with Ktor, Koin, and Exposed, running on a PostgreSQL database. Everything is deployed and managed on Google Cloud.
